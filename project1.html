Project 1

Project 1: Develop Your Team’s “lab book”

Goal

Learn how to design, develop, and usability test a website using tools of students' choice. Every CS student should be able to develop a usable website. In this project, students will create a website that contains information about the three projects for this class. The first project is to create a “lab book” website for two types of users: (1) students who do programming projects for this course, and (2) the instructor and other students who review the progress and results of the programming project. Students record information about each activity they perform while doing the programming project.

Usability test script



Thank you for agreeing to perform this usability test. Note that we are testing how easy it is to use the “lab book” application. This application describes the activities and experiences of students as they develop a user interface and the corresponding application. We are not testing you. We are interested in discovering what problems you encounter while using this app. Don’t worry about making mistakes. They will help us improve the “lab book” application.

Summary

  • Users mostly like the simplicity of example application and how it was easy to use. The other thing that was interesting to them was the appearance.
  • We realized that every individual has his or her own angle of seeing the project which can help us see the projects in different perspectives. They came up with subtle different ideas for a specific problem which teach us how it is important to have idea of different people and learn from them.
  • Documenting each step of the process helps our workflow organized and easier to review or brainstorm ideas/design whenever we run into complicts which easily happen when we come up with new ideas/designs/approaches.
